# NHL Team Details Response

This endpoint answers a specific NHL API question: Team Details. The payload is built around team, seasonsWithEdgeStats, shotSpeed, skatingSpeed, distanceSkated, sogSummary, sogDetails, and zoneTimeDetails. Repeating sections include seasonsWithEdgeStats (5), sogSummary (4), and sogDetails (17). seasonsWithEdgeStats rows include id, and gameTypes; sogSummary rows include locationCode, shots, shotsRank, shotsLeagueAvg, shootingPctg, and shootingPctgRank.
