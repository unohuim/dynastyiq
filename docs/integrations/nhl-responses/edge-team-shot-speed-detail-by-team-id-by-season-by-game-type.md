# NHL Team Shot Speed - Detail Response

This endpoint answers a specific NHL API question: Team Shot Speed - Detail. The payload is built around hardestShots, and shotSpeedDetails. Repeating sections include hardestShots (10), and shotSpeedDetails (3). hardestShots rows include player, gameCenterLink, gameDate, gameType, isHomeTeam, and shotSpeed; shotSpeedDetails rows include position, topShotSpeed, avgShotSpeed, shotAttemptsOver100, shotAttempts90To100, and shotAttempts80To90.
